# Каталог ігор

Консольна утиліта для ведення особистого каталогу відеоігор.

## Запуск

```bash
python main.py
```

## Вимоги

- Python 3.10+
- Зовнішні бібліотеки не потрібні

## Функції

- Додати гру (назва, платформа, статус, оцінка)
- Переглянути каталог
- Редагувати гру
- Видалити гру
- Фільтрувати за статусом
- Переглянути статистику

## Файли

- `main.py` — основний файл програми
- `data.json` — файл даних (створюється автоматично при першому запуску)

## Формат даних

```json
[
  {
    "Name": "Valheim",
    "Platform": "PC",
    "Status": "граю",
    "Rating": 10
  }
]
```
